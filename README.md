# رفيق المسلم (GitHub Pages)

**Build:** 20260328-191035

## ما الذي يتضمنه الموقع؟
- مواقيت الصلاة بطريقة **أم القرى** (method=4) + **العصر حنبلي/جمهور** (school=0).
- عرض الأوقات بنظام **12 ساعة**.
- إضافة: **الضحى** (الشروق + 15 دقيقة) و**الثلث الأخير من الليل** (من المغرب إلى فجر الغد).
- اتجاه القبلة (حساب مسار أعظم) + بوصلة عبر حساسات الجهاز.
- أذكار مختارة (صباح/مساء/نوم/دبر الصلاة/متفرقة) من ملف JSON.
- صفحة روابط علمية موثوقة (قرآن/حديث/فتاوى).
- دعم PWA + خدمة عامل Service Worker مع كاش متغير حسب البناء لتجنب مشكلة التحديث.

## النشر على GitHub Pages
- ارفع **محتويات** هذا المجلد إلى جذر المستودع `Islam`.
- فعّل Pages من Settings → Pages على الفرع الصحيح ومسار `/root`.

## تعديل رقم الإصدار
- افتح `assets/js/config.json` وعدّل `version` كما تريد.
