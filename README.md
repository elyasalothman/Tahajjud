
# رفيق المسلم (نسخة GitHub Pages)

- **إصدار الواجهة:** 0.0.0 (غيّر الرقم عند الحاجة من `assets/js/config.js`).
- **العرض الزمني:** 12 ساعة (ص/م).
- **طريقة الحساب:** أم القرى (method=4)، **مذهب العصر:** جمهور/حنبلي (school=0).
- **ميزات مضافة:** الضحى، الثلث الأخير، صفحة روابط مفيدة، بوصلة القبلة، دعم PWA.

## النشر على GitHub Pages
1) ارفع الملفات إلى الفرع `gh-pages` أو فعّل Pages من إعدادات المستودع والمسار الجذر.
2) إن كان موقعك ضمن مسار فرعي، أبقِ المسارات **نسبية** كما هي.

## ملاحظات
- المواقيت من واجهة AlAdhan العامّة.
- الضحى: الشروق + 15 دقيقة حتى قبل الظهر بـ 10 دقائق (قابلة للتعديل في `config.js`).
- الثلث الأخير: من حساب المدة بين مغرب اليوم وفجر الغد.
